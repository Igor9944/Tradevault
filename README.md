# TradeVault PRO

Journal de trading professionnel — traders sérieux & challengers Prop Firm.

## Stack
React 18 · TypeScript · Vite · Tailwind CSS v4 · Supabase · Vercel

## Développement local

\`\`\`bash
npm install
cp .env.example .env.local
# Remplir VITE_SUPABASE_URL et VITE_SUPABASE_ANON_KEY
npm run dev
\`\`\`

## Variables d'environnement (Vercel → Settings → Environment Variables)

| Variable | Description |
|---|---|
| `VITE_SUPABASE_URL` | URL du projet Supabase |
| `VITE_SUPABASE_ANON_KEY` | Clé publique anon |
| `SUPABASE_SERVICE_KEY` | Clé service role (serveur uniquement) |
| `RESEND_API_KEY` | Clé API pour les emails |
