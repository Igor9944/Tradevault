# TradeVault PRO

Journal de trading professionnel — traders sérieux & Prop Firm.

## Stack
React 18 · TypeScript · Vite · Tailwind CSS v4 · Supabase · Vercel

## Dev local
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Env vars requises (Vercel → Settings → Environment Variables)
- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_KEY
- RESEND_API_KEY
